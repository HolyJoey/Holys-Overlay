# UI-Holy
LUA script for Stand

Features:
Enable a playerwindow displaying the following:
- Name
- Rank
- Modder Status
- Attacker Status
- Language
- Input Device
- Vehicle

Enable an window displaying info such as:
- Host
- Script Host
- Current Time (IRL)
- FPS
- Ped Pool
- Vehicle Pool
- Objes Pool
- Pickup Pool
