# Holy's Overlays
LUA script for Stand

Features:

Enable a player overlay displaying the following:
- Name
- Rank
- Modder Status
- Attacker Status
- Language
- Input Device
- Boss
- Vehicle

Enable an info overlay displaying information like:
- Host
- Script Host
- Players
- Modders
- Friends
- Strangers
- Current Time (IRL)
- FPS
- Ped Pool
- Vehicle Pool
- Objes Pool
- Pickup Pool
